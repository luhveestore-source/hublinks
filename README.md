# ✨ Luhvee Stores - Centralizador de Links

Este é o repositório oficial da página de links da **Luhvee Stores**, desenvolvida em Python utilizando a biblioteca **Streamlit**. 

O objetivo deste projeto é fornecer uma interface rápida, elegante e estável para que os clientes encontrem todos os canais oficiais da loja em um só lugar.

## 🚀 Como este projeto foi feito
* **Linguagem:** Python
* **Framework:** Streamlit
* **Hospedagem:** Streamlit Community Cloud
* **Design:** Custom CSS (Dark Theme & Gold)

## 📁 Estrutura de Arquivos
* `app.py`: Código principal com a interface e links.
* `requirements.txt`: Lista de dependências necessárias para o servidor.
* `1000396187.jpeg`: Logotipo oficial da loja.
* `README.md`: Documentação do projeto.

## 🛠️ Como atualizar os links
1. Abra o arquivo `app.py`.
2. Procure pela função `render_card`.
3. Altere as URLs para os novos links desejados.
4. Salve e faça o *commit* no GitHub; o site atualizará automaticamente.
